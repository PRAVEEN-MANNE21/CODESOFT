IRIS FLOWER CLASSIFICATION PROJECT

This project is part of CodSoft Internship (Data Science).

Objective:
To classify iris flowers into three species using machine learning.

Steps followed:
1. Loaded iris dataset from sklearn
2. Converted dataset into DataFrame
3. Converted labels into flower names
4. Split dataset into training and testing sets
5. Trained Logistic Regression model
6. Evaluated model accuracy
7. Tested model on new input data

Result:
Model achieved high accuracy on test data.

Tools used:
- Python
- Pandas
- Scikit-learn
