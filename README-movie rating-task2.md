# Movie Rating Prediction

## Objective
Predict movie ratings using machine learning based on movie genre, director, and actors.

## Dataset
IMDb India Movies Dataset

## Technologies Used
- Python
- Pandas
- Scikit-learn

## Steps Performed
1. Loaded movie dataset
2. Cleaned missing values
3. Encoded categorical features
4. Split data into training and testing sets
5. Trained a Random Forest Regressor
6. Predicted movie ratings
7. Evaluated model using Mean Absolute Error

## Result
The model successfully predicted movie ratings with a Mean Absolute Error close to 1.
